功能描述
================================================================================
本样例展示Rtc模块时钟误差补偿及1Hz输出功能。

说明：
本样例设置Rtc为24小时时间格式，通过外部高速晶振测试XTAL32的时钟频率计算误差，并
设置端口PC13为1Hz输出端口，配置均匀补偿的方式，配置1秒周期中断，LED0每秒闪烁。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）将示波器连接到PC13端口，测量频率为1Hz；
4）同时观察到LED0持续闪烁。
