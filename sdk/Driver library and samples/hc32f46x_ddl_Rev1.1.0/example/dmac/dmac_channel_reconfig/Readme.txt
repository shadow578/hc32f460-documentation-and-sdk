功能描述
================================================================================
本样例为如何配置DMAC为通道重置模式，通过按键SW2触发通道重置，通过软件触发DMA传输数据。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）全速运行，按下SW2（触发通道重置）；
3）可观察到LED2(绿灯)亮。
