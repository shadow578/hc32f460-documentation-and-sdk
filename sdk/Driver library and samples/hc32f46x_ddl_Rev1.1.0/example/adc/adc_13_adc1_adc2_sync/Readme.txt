功能描述
================================================================================
1）例程实现了 ADC1 和 ADC2 同步模式的配置和用法；
2）ADC 同步工作的触发源可选择内部事件或外部引脚（PB7），本例程可选 Timer0
   EVT_TMR02_GCMA 事件和 PB7 作为触发源。

================================================================================

---------------------

---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动 IDE 的调试功能，断点查看程序运行情况；
3）如果选择 PB7 作为触发源，拉低 PB7 可触发 ADC 扫描。
