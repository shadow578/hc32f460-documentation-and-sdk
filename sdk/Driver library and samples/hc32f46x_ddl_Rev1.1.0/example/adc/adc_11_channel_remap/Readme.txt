功能描述
================================================================================
1）实现了 ADC1 和 ADC2 通道重映射的用法；
2) 例程中，将引脚 ADC12_IN10 映射到 ADC1 的通道 0、1、2 和 4，将引脚 ADC1_IN1 映射
   到 ADC1 的通道 15；将引脚 ADC12_IN4 映射到 ADC2 的通道 7；
3）可用串口（波特率 115200）调试助手查看 ADC1 的通道 0、1、2 和 4 的 ADC 值和电压值，
   它们的 ADC 值和电压值是相同的。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）Micro USB 插入 J1 USBDBG 将开发板与电脑相连；
3）启动 IDE 的调试功能，断点查看程序运行情况；或全速运行用串口调试助手查看 ADC 值
   和对应的电压值。

注意
================================================================================
1）ADC 的参考电压来自引脚 VREFH。
