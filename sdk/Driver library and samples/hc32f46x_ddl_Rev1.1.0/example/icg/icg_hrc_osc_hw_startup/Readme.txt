功能描述
================================================================================
本样例展示ICG模块中HRC功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f46x_icg.h中的宏定义。

本样例设置硬件启动HRC，通过PE0端口输出时钟频率，可连接示波器查看；并每秒闪烁LED0
提示工作状态。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）LED0闪烁；
4) 连接示波器到PE0端口，查看输出的HRC时钟频率为16Mhz。
