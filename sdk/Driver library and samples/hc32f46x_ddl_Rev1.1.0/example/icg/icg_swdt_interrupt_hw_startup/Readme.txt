功能描述
================================================================================
本样例展示ICG模块中SWDT中断功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f46x_icg.h中的宏定义。

本样例设置硬件启动SWDT，并设置约1.6秒触发一次SWDT计数溢出中断，通过SW2循环切换工
作模式（正常模式、睡眠模式、停止模式），正常模式LED0闪烁，睡眠模式LED1闪烁，停止
模式LED0和LED1同时闪烁。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）复位按键复位芯片，进入正常模式LED0闪烁；
4）按下SW2切换工作状态到睡眠模式，LED1闪烁；
5）再次按下SW2切换工作状态到停止模式，LED0和LED1闪烁；
6）再次按下SW2切换工作状态到正常模式，LED0闪烁；
