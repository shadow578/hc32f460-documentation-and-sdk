================================================================================
使用步骤
================================================================================
1）打开工程，打开line99的宏定义“#define  __PRINT_TO_TERMINAL”，并重新编译；
2）启动IDE的下载和调试功能，打开菜单view-Terminal I/O，全速运行；
3）随意按下SW6~SW14，观察LED状态以及Terminal I/O窗口的打印信息。
