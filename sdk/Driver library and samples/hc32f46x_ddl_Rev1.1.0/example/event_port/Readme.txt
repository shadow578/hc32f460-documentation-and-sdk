功能描述
================================================================================
本样例为Event Port样例。
样例中，使用EventPort4.3/4.4触发EventPort4事件，再由EventPort4触发EventPort2事件，
并从EventPort2.5输出驱动LED2点亮；同时也提供了EventPort2被触发后的端口（EventPort2.6）
状态读取示例。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下SW2，SW4来产生EventPort4事件；
4）观察由EventPort2.5所控制的LED2变化。

================================================================================
注意
================================================================================
如必要，打开line66 #define  __PRINT_TO_TERMINAL启动IAR中的Terminal I/O打印功能。
