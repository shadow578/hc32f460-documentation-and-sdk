功能描述
================================================================================
本样例主要展示MCU的EMB模块的GPIO输入刹车功能。
说明：
EMB2用于Timer4，刹车端口为PB12/PE15/PA06
（EMB2-4用于Timer4）
EMB2中断内设置软件刹车，清除PortIn刹车标志
在while（1）内做刹车处理，处理完后使能PWM输出

================================================================================
使用步骤
================================================================================
1）示波器查看J6-40(PE09)。
2）J8-51(PB12)连接VCC。
3）下载并运行。
4）J8-51连接GND，J6-40输出变为低电平，LED0(D23,红灯)点亮。
5）J8-51连接Vcc(3V3)，约10秒后，J6-40恢复输出PWM波形，LED0熄灭。
6）重复步骤4）~步骤5）
