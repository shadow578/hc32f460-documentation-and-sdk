功能描述
================================================================================
本样例以XTAL为MPLL时钟源，设置MPLL达到168MHz，并设置系统时钟分频，切换MPLL为系统时
钟，并输出MPLLP/8（八分频21MHz）。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）连接PA8与示波器
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行，停在while（1）；
4）查看变量stcClkFreq，各时钟频率与预期是否相符。
5）查看示波器，MPLLP时钟频率与预期是否相符。
