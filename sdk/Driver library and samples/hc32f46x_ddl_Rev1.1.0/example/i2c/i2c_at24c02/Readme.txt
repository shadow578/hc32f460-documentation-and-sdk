功能描述
================================================================================
本样例为使用I2C外设读写AT24C02 E2PROM的样例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2) 确认开发板上E2PROM硬件连接是否正确
3）启动IDE的下载和调试功能，全速运行
4) 如果E2PROM读写成功，测试板绿灯闪烁，如果读写失败，测试板红灯闪烁
