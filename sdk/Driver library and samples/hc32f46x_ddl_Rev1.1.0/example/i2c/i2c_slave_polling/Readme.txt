功能描述
================================================================================
本样例为I2C外设从机模式进行查询方式的数据收发样例


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）准备另一个测试板下载样例i2c_master_polling。
3）连线：主机SDA->从机SDA；主机SCL->从机SCL；主机GND->从机GND。
4）启动从机板子IDE的下载和调试功能，全速运行；
5）全速运行主机板子的i2c_master_polling样例。
6）从机进行完数据的接收和发送后，绿灯闪烁
7）主机接受到从机返回的正确数据后，绿灯闪烁，如主机通信失败，红灯闪烁。


================================================================================
注意
================================================================================
1）本样例需要两个测试板并且与样例i2c_master_polling配合使用
