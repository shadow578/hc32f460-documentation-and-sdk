---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）在第二个EFM_SectorErase处打断点；
4）查看memory地址（0x0007C000），可观察到所写数据已经写进去；
5）单步运行，可看到该地址数据被擦除。
