功能描述
================================================================================
本样例利用SW4切换芯片模式，并唤醒


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行，可观察到红灯闪烁两下；
3）按下SW4，芯片进入sleep模式；
4）按下SW4，芯片被唤醒，红灯闪烁；
5）按下SW4，芯片进入stop模式；
6）按下SW4，芯片被唤醒，绿灯闪烁；
7）按下SW4，芯片进入powerdown模式；
8）连接GND与NMI管脚（PB11），唤醒芯片；
9）可观察到红灯闪烁两下，芯片被唤醒。
