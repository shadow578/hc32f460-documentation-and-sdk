功能描述
================================================================================
本样例为芯片进入power down模式2，通过NMI（或WKUP00）唤醒


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下SW2，可观察到红灯闪烁两下，然后芯片进入掉电模；
4）连接NMI管脚（PB11）和GND，唤醒芯片；（用wkup00事件，就连接PA0和VCC）
5）按下SW2，可观察到红灯闪烁（表明芯片已被唤醒，继续执行用户代码）。
