功能描述
================================================================================
本样例为芯片进入stop模式，通过EIRQ4（SW4）和EIRQ5（SW3）唤醒


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下SW2，芯片进入stop模式；
4）按下SW3，可看到绿灯亮，芯片唤醒。
5）按下SW4，可看到红灯亮，芯片唤醒。


================================================================================
注意
================================================================================
SW4中断优先级高于SW3，按下SW3后立刻按下SW4，可观察到红灯、绿灯都亮.
