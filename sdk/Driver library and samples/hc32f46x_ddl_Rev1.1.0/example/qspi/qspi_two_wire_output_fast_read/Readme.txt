功能描述
================================================================================
本样例主要展示QSPI两线式输出快速读功能。

说明：
本样例设置QSPI两线式输出快速读模式，每按一次SW2则触发一次擦除flash、写入数据到
flash、从flash读取数据，再讲读写的数据进行比较，相同的LED1亮，不同则LED0亮，每次
按键都将flash地址偏移到下一个扇区。

无

---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）按下SW2触发flash擦除、写入、读出后比较功能；
4）比较结果相同则LED1亮，不同则LED0亮；
5）重复按下SW2改变flash读写地址，再执行3、4的操作。
