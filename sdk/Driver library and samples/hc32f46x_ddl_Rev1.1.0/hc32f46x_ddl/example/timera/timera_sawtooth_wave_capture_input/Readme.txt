功能描述
================================================================================
本样例展示Timera模块捕获及外部中断事件触发捕获功能。

说明：
本样例设置Timera为锯齿波、增计数模式，Timera单元1的通道1、2开启捕获上升沿输入功
能及捕获中断，同时使能SW4触发通道1、2捕获功能，通道1捕获PE09的输入则切换LED0状
态，通道2捕获PE11的输入则切换LED1状态。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
跳线一根

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）通过跳线将PE09和PE11引脚交替拉高，观察LED0和LED1状态；
4）PE09的输入则切换LED0状态，PE11的输入则切换LED1状态，则功能正常；
5）按下SW4按键则触发通道1、2捕获，同时改变LED0、LED1状态。
