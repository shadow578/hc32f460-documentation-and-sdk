功能描述
================================================================================
本样例展示Timera模块PWM输出及外部中断触发Timer启动功能。

说明：
本样例设置Timera为三角波计数模式，Timera单元1的通道1、3配置为PWM输出模式，通过按
键SW4启动Timera输出，启动成功后LED0会闪烁，通过按键SW2改变输出占空比，可以将示波
器连接到PE09（通道1）、PE13（通道3）观察输出波形。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）按键SW4启动Timera的PWM输出功能，观察LED0闪烁；
4）将示波器连接到PE09、PE13观察输出波形，反复触发按键SW2改变输出占空比；
5）观察示波器波形变化。
