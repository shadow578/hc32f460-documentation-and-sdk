功能描述
================================================================================
本样例主要展示MCU的TIMER62的PWMA端口的捕获输入功能(单缓存功能）
说明：
配置timer6，hclk设置为168M，PLCK0为168M
Timer61和Timer62的计数时钟为PCLK0/1024
================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） Timer61的PWMA引脚（PE09）交替输出2种占空比的PWM波形（比较值分别为0x2000和0x6000，周期为0x8000）
3） 用引线连接Timer62的PWMA引脚（PE11）
4)  在live watch窗口可看到u16CaptureA，u16CaptureB的值交替变化，
    占空比ftCapDuty在0.25（25%）和0.75（75%）之间切换
    且缓存传递方向为u16CaptureA-->u16CaptureB
================================================================================
注意
================================================================================
1)为方便观察，livewatch窗口刷新时间建议为100ms。
