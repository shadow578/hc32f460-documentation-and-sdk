功能描述
================================================================================
本样例主要展示MCU的TIMER62的PWMA端口的捕获输入功能(无缓存功能）
说明：
配置timer6，hclk设置为168M，PLCK0为168M
Timer61和Timer62的计数时钟为PCLK0/1024
================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV-HC32F460-LQFP100-050-V1.1

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） Timer61的PWMA引脚（PE09）交替输出3种占空比的PWM波形（比较值分别为0x2000,0x4000,0x6000，周期为0x8000）
3） 用引线连接Timer62的PWMA引脚（PE11）
4)  在live watch窗口可看到u16CaptureA的值交替变化，
且值与timer61的比较值一致
